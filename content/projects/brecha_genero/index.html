Si no educamos a las niñas de nuestro país, cómo esperamos que puedan emanciparse y defenderse de un problema sistémico como el machismo?

Mi intención es usar datos abiertos para explorar la diferencia entre mujeres y hombres respecto a los indicadores de educación.

Para este proyecto utilicé datos que se encuentran libres y disponibles en la página datosabiertos.bo.

El proyecto se encuentra abierto en un repositorio público de github. Los datos crudos se encuentran en la carpeta 00_datos_crudos y el código para limpiar los datos se encuentra en el archivo 01_R, limpieza_datos.R, guardé los datos ya procesados en el archivo 02_datos_limpios.

El código de este y otros gráficos de indicadores relacionados se encuentran descritos en el siguiente post
